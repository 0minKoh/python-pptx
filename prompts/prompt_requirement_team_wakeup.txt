위는 크롤링한 찬양 가사들입니다. 이 가사를 PPT 슬라이드에 적합한 분량으로 나누어주세요.
각 슬라이드는 가독성을 고려하여 반드시 3~4줄 이내의 가사로 이루어져야 하며, 1줄에 공백 포함 최대 15자 이내여야 합니다.
또한 의미가 자연스럽게 끊기는 지점에서 슬라이드 페이지를 분리해야 합니다.
결과는 아래와 같은 json 형태로 반환해주세요.

가사:
```json
[
  {
    "title": "찬양1 제목",
    "artist": "찬양1 아티스트",
    "lyrics": "찬양 가사"
  }
]
```

출력 형식:
```json
[
  {
    "title": "찬양1 제목",
    "artist": "찬양1 아티스트",
    "splitted_lyrics": ["첫 번째 슬라이드 가사", "두 번째 슬라이드 가사", ...]
  }
]
```
